# Тест-кейсы для проекта Zoo Management System
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------
| ID  | Описание теста                         | Входные данные                        | Ожидаемый результат                    | Фактический результат       | Статус  |
| TC1 | Проверка регистрации нового животного  | Имя животного, возраст, вид           | Новый питомец добавлен в систему       | Проверено: добавлен питомец | Пройден |
| TC2 | Поиск животного по имени               | Имя животного                         | Система находит животное по имени      | Проверено: животное найдено | Пройден |      
| TC3 | Редактирование данных животного        | Имя животного, новые данные           | Данные обновлены                       | Проверено: данные обновлены | Пройден |      
| TC4 | Удаление животного                     | Имя животного                         | Животное удалено из системы            | Проверено: питомец удален   | Пройден |      
| TC5 | Генерация отчета о питомцах            | Фильтрация по типу животного          | Сформирован отчет с информацией        | Проверено: отчет корректен  | Пройден |
| TC6 | Оценка удобства интерфейса             | Навигация по системе                  | Интерфейс интуитивно понятен и удобен  | Проверено: интерфейс удобен | Пройден |
