# Zoo Management System 
Простое консольное приложение на Java для управления виртуальным зоопарком. Программа позволяет управлять виртуальным зоопарком: добавлять животных, просматривать список существующих животных, сортировать их по категориям и слушать звуки, которые издают животные. Проект предназначен для студентов, которые изучают Java
## Установка и запуск
Скачивание проекта 
Склонировать репозиторий или скачать архив
git clone https://github.com/USERNAME/zoo-management-system.git
Открыть проект в Inteligj IDEA
Для запуска мы переходим в директорию
Находим файл и запускаем
## Как запустить программу:
1. скачайте все файлы из папки src/Zoo/
2. откройте в программе для программирования (IntelliJ IDEA, Eclipse)
3. найдите файл main.java и нажмите run
## Что умеет программа:
добавлять животных в зоопарк
показывать всех животных
животные издают звуки
сортировка по категориям
## Пример использования 
Добавить животное 
Показать всех животных
Послушать звуки животных
Сортировать по категориям животных
Выйти
Как происходит добавления животного, что просят у пользователя заполнить для добавления животного:
введите тип животного: млекопитающие
введите имя животного: Кузя
введите возраст животного: 3
Готово: животное довалено
