# План тестирования для проекта Zoo Management System
## Цели тестирования
- Функциональное тестирование: проверка всех функциональных компонентов системы, таких как регистрация животных, поиск, управление данными и отчетность.
- Пользовательское тестирование: оценка удобства использования интерфейса, навигации и общего восприятия системы пользователями.
## Тестируемые функциональные компоненты
1. Регистрация животных.
2. Поиск и фильтрация животных.
3. Управление данными о животных (добавление, редактирование, удаление).
4. Генерация отчетов.
5. Интерфейс пользователя: навигация, удобство использования.
## Типы тестирования
1. Функциональное тестирование:
   проверка корректности работы каждой функции.
2. Пользовательское тестирование:
   оценка удобства интерфейса.
   сбор отзывов о простоте и логичности интерфейса.
## Среда тестирования
- Операционная система: windows
- Браузеры: google Chrome, microsoft Edge, yandex
- Сервисы для сбора обратной связи: google Forms, yandex опрос
